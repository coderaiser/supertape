# @supertape/loader-jsx [![NPM version][NPMIMGURL]][NPMURL]

📼[`Supertape`](https://github.com/coderaiser/supertape) loader adds support of jsx.

## Install

```
npm i supertape @supertape/loader-jsx
```

## Usage

```
supertape --loader @supertape/loader-jsx
```
