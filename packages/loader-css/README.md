# @supertape/loader-css [![NPM version][NPMIMGURL]][NPMURL]

📼[`Supertape`](https://github.com/coderaiser/supertape) loader adds support of css.

## Install

```
npm i supertape @supertape/loader-css
```

## Usage

```
supertape --loader @supertape/loader-css
```
