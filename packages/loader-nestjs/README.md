[NPMURL]: https://npmjs.org/package/@supertape/loader-nestjs "npm"

📼[`Supertape`](https://github.com/coderaiser/supertape) loader adds support of ts.

## Install

```
npm i supertape @supertape/loader-nestjs
```

## Usage

```
supertape --loader @supertape/loader-nestjs
```
